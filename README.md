# Marine Track

MVP-система для автоматического обнаружения судов и кильватерных следов по открытым спутниковым данным.

Проект начинается с воспроизводимого физико-геометрического конвейера, а не с преждевременного обучения нейросети. Базовый сценарий: найти оперативные Sentinel-1/Sentinel-2 сцены по AOI, обнаружить кандидаты судов, выделить след, оценить координаты, курс и, только при наличии валидного признака, скорость.

## Приоритет источников

1. Sentinel-1 SAR через NASA ASF (`asf_search`). Основной источник для круглосуточной и всепогодной детекции судов/следов.
2. Sentinel-1/Sentinel-2 через Copernicus Data Space Ecosystem STAC/OData/Sentinel Hub.
3. Sentinel-2 optical/MSI через STAC/Sentinel Hub для белой воды, альбедо, оптического следа и оценки скорости по межканальному смещению.
4. Copernicus Marine Toolbox для вспомогательных полей: течение, ветер, волны, SST.
5. AIS-источники для валидации: локальный AIS, NOAA MarineCadastre, Global Fishing Watch public datasets.

Если источник недоступен, система должна переходить к следующему по приоритету.

## Что реализуется в MVP

- CLI-каркас `marine-track`.
- Конфигурация источников и fallback-цепочек.
- Абстракция `SceneProvider`.
- Провайдеры ASF и STAC-каталогов.
- Модели данных: сцена, детекция, валидация.
- Геометрические расчеты: курс, расстояние, скорость по смещению, скорость по Kelvin wake.
- Скелет детекции: CFAR/пороговая SAR-детекция, Hough/Radon wake-детекция как следующий слой.
- Вывод GeoJSON.
- Тесты для расчетов и fallback-логики.

## Научная база

См. `docs/TECHNICAL_SPEC.md` и `docs/SCIENCE_BASIS.md`.

Ключевые принципы:

- SAR wake detection опирается на линейность компонент кильватерного следа и методы Radon/Hough/sparse regularization.
- Sentinel-2 может использовать межканальное смещение движущегося судна для оценки скорости.
- Kelvin wake speed считается только при измеримой длине волны и прохождении физических проверок.
- Скорость не оценивается, если нет валидного метода; в выходных данных ставится `not_estimated`.

## Быстрый старт

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .[dev]
marine-track --help
```

Пример запуска:

```bash
marine-track run \
  --aoi data/aoi/example_black_sea.geojson \
  --from 2026-07-01T00:00:00Z \
  --to 2026-07-06T00:00:00Z \
  --sensor auto \
  --output runs/black_sea_20260706
```

## Статус

Текущий статус: первичный MVP-каркас. Он фиксирует архитектуру, источники данных, fallback-доступ, расчетные формулы и интерфейсы. Следующий шаг — реализация обработки реальных Sentinel-1 GRD/RTC и Sentinel-2 L2A сцен.
