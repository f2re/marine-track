# UX/UI аудит Telegram-бота Marine Track

## Цель пользователя

Пользователь хочет быстро понять, есть ли свежий спутниковый снимок по интересующей акватории, запустить детекцию судов и получить пригодные артефакты: обзор, crop судов, GeoJSON/CSV/Parquet и отчет.

## Найденные проблемы

1. `/start` не вел к цели. Пользователь видел список команд и должен был помнить синтаксис.
2. Основной сценарий требовал token: сначала `/dates`, потом кнопка или `/detect token`. Это техническая модель, не пользовательская.
3. Не было главного меню и возврата в безопасную точку навигации.
4. `/status` был техническим dump-ом и не показывал, что мешает работе: AOI, land mask, доступ, output dir.
5. Ошибки не давали следующего действия.
6. Для default AOI не было одной кнопки “найти и обработать”.
7. Результаты детекции не возвращали пользователя в дальнейшие действия.

## Реализованные изменения

### Главное меню

Добавлен модуль `telegram_ui.py` с единым inline-меню:

- `🔎 Найти суда` — основной сценарий: свежая сцена по default AOI и запуск детекции;
- `🕒 Сроки снимков` — показать доступные сцены;
- `⚙️ Статус` — человекочитаемый статус;
- `❓ Помощь` — краткая инструкция;
- `🆔 Мой ID` — быстрый id для настройки `TELEGRAM_ADMIN_IDS`.

### Быстрый сценарий default AOI

Добавлен `detect_default_aoi()`:

```text
default AOI → detection-capable scene search → register scene token → run detection → send outputs
```

Пользователь может нажать `🔎 Найти суда` без ручного ввода bbox/token.

### Улучшенные сообщения

- `/start` теперь показывает цель, быстрый путь и текущий default AOI.
- `/help` разделяет сценарии “без ручного ввода” и “точное управление командами”.
- `/status` показывает доступ, AOI, lookback, sensor, land mask, shoreline buffer, output dir.
- Ошибки `/detect`, `/detectbbox` и MaterializationError теперь предлагают вернуться к меню или использовать быстрый сценарий.

### Навигация после результата

После детекции summary-сообщение содержит меню. Пользователь может сразу запустить новый поиск, открыть сроки, статус или помощь.

## Текущий рекомендуемый сценарий

1. `/start`
2. `🔎 Найти суда`
3. Дождаться overview/crops/GeoJSON/CSV/Parquet/report.
4. Для ручной зоны использовать:

```text
/detectbbox sentinel1 west south east north 12
```

## Следующие UX-улучшения

1. Добавить сохраненные пользовательские bbox/AOI через Telegram callbacks.
2. Добавить пагинацию списка сцен, если `max_results > 12`.
